Minimax search in Mancala

Minimax search is a fundamental search algorithm in Artificial Intelligence. The purpose of minimax search is to minimize the maximimum loss in a worst case scenario.

This algorithm can be implemented in n-player games but is most commonly implemented in 2-player games such as: checkers, chess, connect 4, mancala, etc.

Heuristic is a functon that returns an approximate value of how much the maximizing player is winning or losing. Postive numbers are more beneficial for pmax (the player) whereas negative numbers are more benficial for pmin (the player's opponent).
